# Flow Spec: Emotion Logging & Immediate Coaching

## 목적
사용자가 감정을 기록하는 과정 자체가 '치유'와 '자기 객관화'가 되도록 설계한다.

## Flow Steps

### Step 1: Affect Labeling (감정 명명)
* **UI:** 2D Grid (쾌/불쾌, 고각성/저각성)에서 현재 상태를 터치.
* **Logic:** 터치한 좌표에 따라 적절한 감정 단어 후보군 노출 (예: 고각성+불쾌 -> 분노, 불안, 짜증).
* **Brain Science:** 감정에 이름을 붙이는 것만으로도 편도체 활성도가 감소함.

### Step 2: Contextualizing (맥락화)
* **Question:** "지금 어떤 상황인가요?" (What is happening?)
* **UI:** 빠른 태그 선택 (활동, 사람, 장소).
* **Logic:** 자주 쓰는 조합 추천.

### Step 3: Somatic Marking (신체 감각 확인)
* **Question:** "몸에서는 어떤 신호가 느껴지나요?"
* **UI:** 바디 스캔 이미지 (머리, 가슴, 배, 손발 선택).
* **Psychology:** 감정을 머리로만 인지하지 않고 신체 감각과 연결하여 해리(Dissociation) 방지.

### Step 4: Cognitive Analysis (원인 분석 & 가치 탐색)
* **Condition:** 감정 강도가 7 이상인 경우 진입.
* **Happy Case:** "이 순간을 통해 당신의 어떤 가치가 충족되었나요?" (성취, 연결, 평화 등) -> **Savoring(음미하기) 강화**
* **Angry/Sad Case:** "이 상황이 당신에게 중요하게 여기는 무엇을 건드렸나요?" (존중, 정직, 시간약속 등) -> **Triggers(방아쇠) 식별**

### Step 5: Action & Solution (대처)
* **Question:** "이 감정을 어떻게 다루고 싶나요?"
* **Recommendation:** 사용자의 과거 데이터 기반 추천.
    * *Anger Type A (신체형):* "나가서 5분 걷기" 추천.
    * *Anger Type B (사고형):* "감정 일기 쓰기" 추천.
