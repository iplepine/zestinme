# Module: Happiness & Values (Compass & Treasure)

## 1. Overview
행복은 강도가 아니라 빈도이며, '음미하기(Savoring)'를 통해 강화됩니다. 또한, 목표(Goal)와 가치(Value)를 구분하여 삶의 방향성을 설정합니다.

## 2. Feature: 가치관 카드 분류 (Values Card Sort)
*   **방식:** 틴더(Tinder) 스타일의 스와이프 인터페이스.
*   **카드:** 자율성, 가족, 창의성, 건강, 봉사 등.
*   **액션:**
    *   우측: 매우 중요함 (Sunlight).
    *   좌측: 중요하지 않음 (Shade).
    *   위쪽: 보통.
*   **결과:** 상위 3개 핵심 가치가 '나침반(Sun Dial)'에 등록됨.
*   **데일리 퀘스트:** "오늘 당신의 태양은 어디를 비추었나요?"

## 3. Feature: 음미하기 (Savoring)
*   **Trigger:** SAM 척도에서 High Valence 입력 시 작은 **꽃봉오리**가 맺힘.
*   **Quest:** "꽃 피우기 (Blooming)"
    *   꽃봉오리는 아직 닫혀 있음.
    *   활짝 피우기 위해 구체적인 질문에 답해야 함.
    *   "그물에 걸린 햇살(기억)의 색깔은 어땠나요? (시각)"
    *   "이 꽃의 향기를 누구에게 전해주고 싶은가요? (공유)"
*   **Effect:** 긍정적 기억을 '압화(Pressed Flower)' 또는 '열매청(Fruit Preserve)'으로 저장하여 자원화.
