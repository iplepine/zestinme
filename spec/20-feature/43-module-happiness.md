# Module: Happiness & Values (Compass & Treasure)

## 1. Overview
행복은 강도가 아니라 빈도이며, '음미하기(Savoring)'를 통해 강화됩니다. 또한, 목표(Goal)와 가치(Value)를 구분하여 삶의 방향성을 설정합니다.

## 2. Feature: 가치관 카드 분류 (Values Card Sort)
*   **방식:** 틴더(Tinder) 스타일의 스와이프 인터페이스.
*   **카드:** 자율성, 가족, 창의성, 건강, 봉사 등.
*   **액션:**
    *   우측: 매우 중요함.
    *   좌측: 중요하지 않음.
    *   위쪽: 보통.
*   **결과:** 상위 3개 핵심 가치가 '나침반'에 등록됨.
*   **데일리 퀘스트:** "오늘 당신의 항해는 나침반(가치)의 방향과 일치했나요?"

## 3. Feature: 음미하기 (Savoring)
*   **Trigger:** SAM 척도에서 High Valence 입력 시.
*   **Quest:** "보석 닦기"
    *   잡은 물고기가 보석처럼 빛남.
    *   더 밝게 빛나게 하려면 구체적인 질문에 답해야 함.
    *   "그 순간의 감각(시각, 청각)은 어땠나요?"
    *   "이 순간을 누구와 나누고 싶은가요?"
*   **Effect:** 긍정적 기억을 '섬광 기억(Flashbulb Memory)'으로 저장하여 자원화.
