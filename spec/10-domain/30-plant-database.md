## 3. Mystery Plant Visual Archetypes (미스터리 식물 시각적 분류)

감정 → 환경 → 식물로 매핑하기 위해 **“보는 순간 분류는 되지만, 종은 모르는 상태”**를 지향하며 생성된 시각적 원형(Archetype)입니다.

| 코드 | 카테고리 (Visual) | 핵심 실루엣 |
| :--- | :--- | :--- |
| **M-HERB** | 허브형 | 가늘고 부드러운 줄기, 작은 잎 |
| **M-LEAF** | 관엽형 | 넓은 잎, 안정적인 형태 |
| **M-SUCC** | 다육/선인장형 | 두껍고 응축된 볼륨 |
| **M-ORCH** | 난초형 | 비대칭, 공중감, 섬세함 |
| **M-TREE** | 소형목/관목형 | 줄기 중심, 위로 퍼짐 |
| **M-WEIRD** | 이형/신비형 | 비정형, 낮은 밀도 |
