# Core Mechanics: The Fishing Metaphor

## 1. 도파민 보상 회로와 가변 비율 강화 (Variable Ratio Reinforcement)
*   **원리:** 보상이 예측 불가능할 때 도파민 시스템(추구와 기대)이 가장 강력하게 활성화됨.
*   **적용:** 기록(미끼 투척)에 대한 결과를 확률적으로 제공하여, '의무'가 아닌 '기대'로 전환.
*   **기대 오차(Prediction Error):** 무엇이 잡힐지 모르는 불확실성이 지속적인 사용 동기를 부여.

## 2. 핵심 루프 (Core Loop)

### 2.1 미끼 제작 (Input)
사용자의 '자기 기록'이 미끼가 됩니다.
*   **저등급 미끼:** 단순 탭 입력 (SAM 척도 체크).
*   **고등급 미끼:** 텍스트 기록 (감사 일기, 분노 로그).
*   **전설 등급 미끼:** 퀘스트 완료 (7일 수면 프로토콜 완료 등).

### 2.2 캐스팅 (Action)
*   화면을 스와이프하여 낚싯줄을 던지는 행위.
*   물리적 인터랙션을 통해 '행위의 시작'을 명확히 인지.

### 2.3 기다림 (Mindfulness)
*   **의도된 지연:** 찌가 물에 떠 있는 15~60초 동안 즉각적인 반응을 지연시킴.
*   **활동:** 호흡 가이드(들숨 4초, 날숨 6초) 또는 '충동 서핑(Urge Surfing)' 연습.
*   **목적:** 즉각적 만족 지연 능력 향상 및 그라운딩(Grounding).

### 2.4 수집 (Reward)
*   **유형 보상:**
    *   **물고기:** 융기안 원형을 상징하는 수집품.
    *   **데코 아이템:** 선착장을 꾸미는 소품.
    *   **재화:** Sand Dollar.
*   **무형 보상:**
    *   **유리병 편지:** 데이터 기반의 심리적 통찰.
    *   **기억 조각:** 과거의 긍정적 기록 회상 (Savoring).

## 3. 확률형 드롭 시스템 (Weighted Probability Drop)
사용자의 정서 상태(Digital Phenotype)에 따라 드롭 테이블이 동적으로 변경됩니다.

### 3.1 알고리즘 로직
1.  **Biome 선택:** User Mood(Valence, Arousal)에 따라 '화산 지대', '심해', '평온한 바다' 등 결정.
2.  **Luck 보정:** 미끼 품질(기록의 충실도)에 따라 희귀도 보정.
3.  **천장 시스템 (Pity Timer):** 고통스러운 기록이 지속될 경우 '치료적 통찰' 아이템 강제 드롭.
4.  **RNG:** 최종 확률 롤링.
