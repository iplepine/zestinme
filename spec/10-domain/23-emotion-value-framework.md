# Emotion Recording & Value Discovery Framework

## 1. 개요 (Overview)

본 문서는 사용자의 정서 입자도(Emotional Granularity)를 높이고, 감정을 신호(Signal)로 활용하여 핵심 가치(Core Values)를 발견하기 위한 프레임워크를 정의한다.

### 핵심 철학
*   **"All emotions are signals for values."** (모든 감정은 가치에 대한 신호다.)
*   부정적 감정: 가치가 위협받거나 결핍됨.
*   긍정적 감정: 가치가 충족되거나 확장됨.

---

## 2. 3단계 분석 프로세스 (3-Step Analysis Process)

### Step 1: Quadrant Selection (1차 분류)
제임스 러셀(James Russell)의 **정서 원형 모델(Circumplex Model of Affect)**에 기반하여, 에너지(Arousal)와 쾌적함(Valence)의 두 축으로 4분면을 선택한다.

| Quadrant | Color (RULER) | State | Description | Core Question |
| :--- | :--- | :--- | :--- | :--- |
| **RED** | 빨강 | **Stress / Anxiety** | 고에너지 / 불쾌 | "나의 어떤 가치(경계)가 위협받고 있는가?" |
| **BLUE** | 파랑 | **Sadness / Burnout** | 저에너지 / 불쾌 | "나는 무엇을 잃어버렸거나 결핍되었다고 느끼는가?" |
| **YELLOW** | 노랑 | **Energy / Joy** | 고에너지 / 쾌적 | "나를 움직이게 하는 내재적 동기는 무엇인가?" |
| **GREEN** | 초록 | **Calm / Peace** | 저에너지 / 쾌적 | "무엇이 나를 안전하고 충만하게 만드는가?" |

### Step 2: Emotional Granularity (2차 상세 분류)
단순한 기분(Feeling)을 넘어 구체적인 정서(Emotion) 단어를 선택한다.

**[원칙] 다중 선택 (Multiple Selection) 허용**
인간은 상반되거나 복합적인 감정을 동시에 느낄 수 있습니다. (예: 시원섭섭함, 불안하지만 설렘)
*   **복합 정서의 인정:** 감정을 억지로 하나로 단순화하지 않고 있는 그대로 기록합니다.
*   **가치 발견의 단서:** 서로 다른 감정의 조합은 중요한 통찰을 제공합니다.
    *   *예: 슬픔(상실) + 안도감(안전) = "힘든 상황이 끝났지만(안전), 그 과정에서 무언가 잃었음(애착)"*

#### ① RED ZONE (스트레스/높은 에너지/불쾌)
이 영역의 감정은 당신의 **'경계(Boundaries)'**가 침해받았거나 **'통제권'**을 잃었을 때 발생합니다.

| 세부 감정 (Keyword) | 심리적 의미 | 연결된 가치 (Values) |
| :--- | :--- | :--- |
| **화난/분노 (Angry)** | 나의 기준이나 권리가 침해됨. | 정의, 공정성, 존중 |
| **불안한 (Anxious)** | 미래의 불확실성이나 실패에 대한 두려움. | 안전, 준비성, 완벽함 |
| **억울한 (Resentful)** | 내가 한 만큼 대우받지 못함, 경계 침해. | 인정, 호혜성, 경계 유지 |
| **압도된 (Overwhelmed)** | 감당할 수 있는 용량을 초과함. | 균형, 효율성, 질서 |
| **질투나는 (Jealous)** | 소중한 관계나 지위를 잃을까 봐 두려움. | 독점적 관계, 신뢰, 소속감 |
| **짜증난 (Annoyed)** | 사소한 방해로 목표 추구가 지연됨. | 효율, 시간 엄수, 흐름 |

#### ② BLUE ZONE (지침/낮은 에너지/불쾌)
이 영역은 **'상실(Loss)'**이나 **'결핍(Lack)'**을 처리하는 과정입니다.

| 세부 감정 (Keyword) | 심리적 의미 | 연결된 가치 (Values) |
| :--- | :--- | :--- |
| **슬픈 (Sad)** | 소중한 것을 잃음. (중요하지 않으면 슬프지 않음) | 사랑, 애착, 연결 |
| **실망한 (Disappointed)** | 기대했던 기준에 미치지 못함. | 신뢰, 이상(Ideal), 기대 |
| **무기력한/지루한 (Bored)** | 현재 활동에서 의미나 자극을 찾지 못함. | 의미, 성장, 자극 |
| **외로운 (Lonely)** | 사회적 연결의 부재. | 소속감, 친밀감, 유대 |
| **죄책감 (Guilty)** | 자신의 도덕적 기준을 어김. (선한 사람임을 증명) | 도덕성, 책임감, 정직 |
| **부러운 (Envious)** | 타인이 가진 것을 나도 원함. | 야망, 성취, 자기 발전 |

#### ③ YELLOW ZONE (활기참/높은 에너지/쾌적)
이 영역은 당신의 **'욕망'**이 충족되고 **'확장'**되는 상태입니다.

| 세부 감정 (Keyword) | 심리적 의미 | 연결된 가치 (Values) |
| :--- | :--- | :--- |
| **신나는 (Excited)** | 기대되는 보상이나 경험이 있음. | 모험, 새로운 경험, 변화 |
| **자랑스러운 (Proud)** | 나의 노력으로 사회적 지위나 인정을 획득함. | 성취, 탁월함, 인정 |
| **영감 받은 (Inspired)** | 새로운 아이디어나 비전과 연결됨. | 창의성, 비전, 통찰 |
| **열정적인 (Enthusiastic)** | 어떤 대상에 깊이 몰입하고 에너지를 쏟음. | 몰입, 헌신, 목적 |
| **호기심 찬 (Curious)** | 새로운 것을 알고 싶고 탐구하고 싶음. | 지식, 학습, 성장 |
| **유쾌한 (Amused)** | 상황의 모순을 즐기거나 긴장이 해소됨. | 유머, 즐거움, 자유 |

#### ④ GREEN ZONE (차분함/낮은 에너지/쾌적)
이 영역은 당신이 **'안전'**하며 현재에 **'만족'**하는 상태입니다.

| 세부 감정 (Keyword) | 심리적 의미 | 연결된 가치 (Values) |
| :--- | :--- | :--- |
| **편안한 (Relaxed)** | 위협이 없고 긴장이 풀린 상태. | 안전, 휴식, 여유 |
| **감사하는 (Grateful)** | 타인이나 세상으로부터 혜택을 받았음을 인지. | 겸손, 관계, 배려 |
| **만족스러운 (Content)** | 더 바랄 것 없이 현재가 충분함. | 수용, 충족감, 단순함 |
| **평온한 (Serene)** | 내면의 갈등 없이 고요함. | 내적 평화, 조화, 균형 |
| **신뢰하는 (Trusting)** | 타인에게 의지해도 안전하다고 느낌. | 믿음, 진정성, 유대감 |
| **사색적인 (Reflective)** | 자신을 돌아보고 깊이 생각함. | 지혜, 자기 인식, 성찰 |

### Step 3: Contextual Logging (3 T's) & Value Tagging
감정의 맥락을 기록하고 가치를 역설계한다.

1.  **Trigger (촉발 요인):** Who, What, Where.
2.  **Thought (자동적 사고):** 감정 순간 스쳐 지나간 생각.
3.  **Tendency (행동 충동):** 무엇을 하고 싶었는가? (공격, 회피, 포옹 등)
4.  **Value Tag (가치 태그):** 위의 정보를 종합하여 도출한 핵심 가치. (예: #공정성, #성장, #연결)

## 3. Coaching Questions (Deferred Value Discovery)

**"Seed First, Nurture Later"**
감정을 처음 기록하는 시점(Seeding)에는 감정의 종류(2차 상세 분류)까지만 입력하고 종료합니다.
이후 일정 시간(Incubation Time)이 지난 후, 감정이 조금 가라앉았을 때 **'양육(Nurturing)'** 단계에서 아래의 코칭 질문을 통해 가치를 발견합니다. 매번 같은 질문이 아닌, 감정에 따라 다양한 질문을 돌아가며 활용합니다.

### 🔴 RED ZONE (스트레스, 분노, 불안) - 무엇이 위협받고 있는가?
이 감정들은 당신의 **경계(Boundaries)**나 원칙이 침해되었음을 알립니다.

#### 분노/짜증 (침해에 대한 반응)
*   "지금 이 상황에서 '이것만은 지켜졌어야 했다'고 생각하는 것은 무엇인가?" (핵심 원칙)
*   "상대방이 나의 어떤 부분을 가볍게 여겼다고 느껴지는가?" (존중의 기준)
*   "내가 만약 이 상황을 그냥 넘긴다면, 내 모습이 어떻게 비춰질까 봐 걱정되는가?" (자아상/평판)
*   "이 분노가 나에게 어떤 행동을 하라고 부추기고 있는가? 그 행동은 진정으로 내가 원하는 결과를 가져오는가?"

#### 불안/두려움 (위협에 대한 반응)
*   "최악의 상황을 상상했을 때, 내가 잃어버릴까 봐 가장 두려운 것은 무엇인가?" (가장 소중한 자산)
*   "내가 통제할 수 없는 것을 통제하려 하고 있지는 않은가?"
*   "이 불안감을 해결하기 위해 지금 당장 내가 할 수 있는 아주 작은 행동 하나는 무엇인가?"
*   "내가 만약 완벽하게 안전하다고 느낀다면, 지금 무엇을 하고 싶은가?" (억눌린 욕구)

#### 질투/시기 (결핍에 대한 반응)
*   "저 사람이 가진 것 중, 솔직히 나도 갖고 싶어서 배가 아픈 것은 정확히 무엇인가?" (숨겨진 야망)
*   "나는 그것을 가질 자격이 없다고 스스로 믿고 있지는 않은가?"
*   "그것을 얻기 위해 나는 어떤 대가를 치를 준비가 되어 있는가?" (진정성 확인)

### 🔵 BLUE ZONE (우울, 슬픔, 지침) - 무엇이 결핍되었는가?
이 감정들은 당신에게 회복과 연결이 필요함을 알립니다.

#### 슬픔/상실
*   "무엇이 내 삶에서 빠져나간 것 같은 느낌이 드는가?"
*   "그것이 없어서 내 삶이 빈껍데기 같다고 느낀다면, 그것은 나에게 어떤 의미였나?" (가치 확인)
*   "지금 이 슬픔을 충분히 느끼지 않고 억누른다면, 나는 무엇을 잃게 될까?" (애도의 필요성)

#### 지침/번아웃
*   "나는 누구의 기대를 만족시키기 위해 이렇게 애쓰고 있는가?"
*   "내가 'No'라고 말하지 못해서 짊어지고 있는 짐은 무엇인가?" (경계 설정)
*   "지금 당장 모든 의무가 사라진다면, 가장 먼저 하고 싶은(혹은 안 하고 싶은) 것은 무엇인가?" (회복의 욕구)
*   "나를 충전시키는 활동과 방전시키는 활동의 비율은 어떠한가?"

#### 소외감/외로움
*   "나는 어떤 종류의 연결을 갈망하고 있는가? (이해받는 것? 함께 하는 것? 소속되는 것?)"
*   "내가 진정으로 나답게 있을 수 있는 관계는 어디인가?"

### 🟡 YELLOW ZONE (활기, 열정, 기쁨) - 무엇을 추구하고 싶은가?
이 감정들은 당신의 강점과 동기가 어디에 있는지 알려줍니다.

#### 열정/흥분
*   "시간 가는 줄 모르고 했던 이 행동의 어떤 요소가 나를 자극했는가?"
*   "이 순간 내가 '살아있다'고 느끼게 해 준 핵심적인 경험은 무엇인가?"
*   "이 에너지를 활용해서 더 확장해보고 싶은 나의 능력은 무엇인가?"

#### 성취감/자부심
*   "오늘의 성과 중 나 스스로 가장 칭찬해주고 싶은 부분은 무엇인가?"
*   "이 성취가 나에게 중요한 이유는 남들의 인정 때문인가, 아니면 나 스스로의 만족 때문인가?"
*   "이 경험을 통해 확인한 나의 강점은 무엇인가?"

#### 영감/호기심
*   "이 새로운 아이디어가 나를 설레게 하는 이유는 무엇인가?"
*   "이 호기심을 따라가면 내 삶의 어떤 부분이 성장할 것 같은가?"

### 🟢 GREEN ZONE (평온, 감사, 만족) - 무엇으로 충분한가?
이 감정들은 당신의 삶을 지탱하는 **기반(Foundation)**을 보여줍니다.

#### 평온/편안함
*   "지금 내 마음을 고요하게 만드는 환경적/심리적 조건은 무엇인가?" (나만의 안식처 조건)
*   "어떤 걱정을 내려놓았더니 이런 평화가 찾아왔는가?"
*   "이 평온함을 지키기 위해 나는 일상에서 무엇을 보호해야 하는가?"

#### 감사/충만함
*   "당연하다고 생각했는데 다시 보니 선물처럼 느껴지는 것은 무엇인가?"
*   "지금 가진 것만으로도 충분하다고 느낀다면, 그 이유는 무엇인가?" (만족의 기준)
*   "이 고마운 마음을 누구에게, 어떻게 표현하고 싶은가?" (관계의 가치)

## 4. 데이터 활용 (From Data to Coaching)

### Value Map (가치 지도)
*   **Vulnerability (취약점):** 반복되는 Red/Blue 패턴 = 타협할 수 없는 핵심 경계.
*   **Motivation (동기):** 반복되는 Yellow 패턴 = 내재적 동기 원천.
*   **Resilience (회복):** 반복되는 Green 패턴 = 나만의 안식처.

### Value-Action Gap (불일치)
*   머리로 중요하다고 생각하는 가치와 실제 감정이 반응하는 가치 사이의 불일치를 포착하여 코칭 포인트로 활용.
