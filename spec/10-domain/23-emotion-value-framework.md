# Emotion Recording & Value Discovery Framework

## 1. 개요 (Overview)

본 문서는 사용자의 정서 입자도(Emotional Granularity)를 높이고, 감정을 신호(Signal)로 활용하여 핵심 가치(Core Values)를 발견하기 위한 프레임워크를 정의한다.

### 핵심 철학
*   **"All emotions are signals for values."** (모든 감정은 가치에 대한 신호다.)
*   부정적 감정: 가치가 위협받거나 결핍됨.
*   긍정적 감정: 가치가 충족되거나 확장됨.

---

## 2. 3단계 분석 프로세스 (3-Step Analysis Process)

### Step 1: Quadrant Selection (1차 분류)
제임스 러셀(James Russell)의 **정서 원형 모델(Circumplex Model of Affect)**에 기반하여, 에너지(Arousal)와 쾌적함(Valence)의 두 축으로 4분면을 선택한다.

| Quadrant | Color (RULER) | State | Description | Core Question |
| :--- | :--- | :--- | :--- | :--- |
| **RED** | 빨강 | **Stress / Anxiety** | 고에너지 / 불쾌 | "나의 어떤 가치(경계)가 위협받고 있는가?" |
| **BLUE** | 파랑 | **Sadness / Burnout** | 저에너지 / 불쾌 | "나는 무엇을 잃어버렸거나 결핍되었다고 느끼는가?" |
| **YELLOW** | 노랑 | **Energy / Joy** | 고에너지 / 쾌적 | "나를 움직이게 하는 내재적 동기는 무엇인가?" |
| **GREEN** | 초록 | **Calm / Peace** | 저에너지 / 쾌적 | "무엇이 나를 안전하고 충만하게 만드는가?" |

### Step 2: Emotional Granularity (2차 상세 분류)
단순한 기분(Feeling)을 넘어 구체적인 정서(Emotion) 단어를 선택한다.

#### RED Zone (위협/침해)
*   **분노 (Angry/Furious):** 공정성, 존중 침해.
*   **불안 (Anxious):** 안전, 통제 상실 예기.
*   **질투 (Jealous):** 숨겨진 야망, 성취 욕구.
*   **압도됨 (Overwhelmed):** 균형, 효율성 상실.
*   **좌절 (Frustrated):** 성장, 유능감 (발전하고 성과를 내는 것을 중요시함).


#### BLUE Zone (상실/고갈)
*   **슬픔 (Sad/Grieving):** 사랑, 연결, 의미 (중요하지 않은 것을 잃었을 때 슬퍼하지 않음).
*   **실망 (Disappointed):** 기대, 신뢰, 이상 (높은 기준이나 비전을 중요시함).
*   **소진된 (Exhausted/Burnout):** 자기돌봄, 지속가능성 (삶의 방식이 가치와 충돌).
*   **외로운 (Lonely/Isolated):** 소속감, 친밀감 (타인과의 깊은 유대 중시).
*   **죄책감 드는 (Ashamed/Guilty):** 도덕성, 책임감 (올바른 행동을 중요시함).

#### YELLOW Zone (확장/구축)
*   **열정 (Excited/Enthusiastic):** 모험, 열정, 변화 (변화와 자극을 중요시함).
*   **영감 (Inspired/Creative):** 창조성, 비전, 미적 가치 (새로운 관점을 얻는 것을 사랑함).
*   **성취감 (Proud/Accomplished):** 성취, 탁월함, 인정 (능력을 발휘하고 결과를 내는 것을 중요시함).
*   **몰입 (Curious/Flow):** 지식, 성장, 학습 (세상을 이해하고 배우는 과정을 즐김).
*   **재미 (Amused/Playful):** 유머, 즐거움, 자유 (삶의 가벼움과 웃음을 중요시함).

#### GREEN Zone (회복/통합)
*   **평온 (Peaceful/Serene):** 내적 평화, 조화, 단순함 (복잡함보다 명료함과 고요함을 선호).
*   **감사 (Grateful):** 관계, 겸손, 영성 (타인과의 긍정적 연결과 삶의 선물을 소중히 여김).
*   **만족 (Content):** 수용, 충족감 (더 많은 것보다 지금 가진 것의 가치를 앎).
*   **안도 (Relieved/Relaxed):** 안전, 휴식 (긴장이 없는 상태와 심리적 안정감을 중요시함).
*   **신뢰 (Loved/Trusting):** 신뢰, 친밀감, 가족 (깊고 안전한 인간관계를 핵심 가치로 둠).

### Step 3: Contextual Logging (3 T's) & Value Tagging
감정의 맥락을 기록하고 가치를 역설계한다.

1.  **Trigger (촉발 요인):** Who, What, Where.
2.  **Thought (자동적 사고):** 감정 순간 스쳐 지나간 생각.
3.  **Tendency (행동 충동):** 무엇을 하고 싶었는가? (공격, 회피, 포옹 등)
4.  **Value Tag (가치 태그):** 위의 정보를 종합하여 도출한 핵심 가치. (예: #공정성, #성장, #연결)

## 3. Coaching Questions (가치 발견 질문)

### Red Zone (위협/침해)
*   **분노/짜증:** "지금 침해당했다고 느끼는 나의 원칙이나 기준은 무엇인가? 상대방이 지키지 않은(혹은 상황이 어긴) 나의 '선'은 무엇인가?"
*   **불안/두려움:** "내가 잃어버릴까 봐 가장 두려워하는 것은 무엇인가? (사람, 평판, 돈 등)" → 그것이 당신이 지키고 싶은 핵심 자산입니다.
*   **시기/질투:** "저 사람이 가진 것 중, 솔직히 나도 갖고 싶은 것은 무엇인가?" (숨겨진 야망과 꿈을 가장 빠르게 찾아줍니다.)
*   **압도됨:** "내가 지금 놓아버리면 큰일 난다고 믿는 것은 무엇인가? 그것이 정말로 내가 책임져야 할 몫인가?"

### Blue Zone (상실/고갈)
*   **슬픔:** "무엇을 잃어버린 느낌이 드는가? 그것이 내 삶에서 왜 그렇게 중요했는가?"
*   **실망:** "나는 어떤 기대를 하고 있었는가? 그 기대는 나에게 어떤 의미가 있는가?"
*   **지침(번아웃):** "나를 고갈시키면서까지 내가 지키려고 애썼던 것은 무엇인가? 그것이 나의 건강보다 중요한가?"
*   **죄책감:** "내가 어겼다고 생각하는 나의 원칙은 무엇인가? 나는 어떤 사람이 되고 싶었는가?"

### Yellow Zone (확장/구축)
*   **열정/흥분:** "이 활동의 어떤 부분이 나를 이렇게 살아있게 만드는가? (사람들과의 만남? 문제 해결? 새로운 정보?)"
*   **자랑스러움:** "나의 어떤 노력이 결실을 맺었는가? 나는 어떤 종류의 성과를 낼 때 가장 뿌듯한가?"
*   **몰입(Flow):** "시간 가는 줄 모르고 했던 이 활동은 나의 어떤 강점(분석력, 공감능력, 손재주 등)을 사용했는가?"
*   **영감:** "이 순간 나에게 어떤 새로운 가능성이 보였는가?"

### Green Zone (회복/통합)
*   **평온:** "무엇이 지금 나를 이렇게 편안하게 만드는가? (조용한 환경? 갈등의 해결? 자연?)" → 당신에게 필요한 최적의 환경을 알려줍니다.
*   **감사:** "오늘 내가 당연하게 여기지 않고 고마워한 것은 무엇인가?" → 당신의 삶을 지탱하는 기반 가치를 보여줍니다.
*   **만족:** "지금 이 순간, 더 바랄 것이 없다고 느끼는 이유는 무엇인가?" → 당신의 욕망이 멈추는 지점, 즉 진정한 행복의 기준을 보여줍니다.

## 4. 데이터 활용 (From Data to Coaching)

### Value Map (가치 지도)
*   **Vulnerability (취약점):** 반복되는 Red/Blue 패턴 = 타협할 수 없는 핵심 경계.
*   **Motivation (동기):** 반복되는 Yellow 패턴 = 내재적 동기 원천.
*   **Resilience (회복):** 반복되는 Green 패턴 = 나만의 안식처.

### Value-Action Gap (불일치)
*   머리로 중요하다고 생각하는 가치와 실제 감정이 반응하는 가치 사이의 불일치를 포착하여 코칭 포인트로 활용.
