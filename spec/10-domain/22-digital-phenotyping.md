# Digital Phenotyping & Data Collection

## 1. 하이브리드 데이터 수집 모델

### 1.1 수동적 데이터 (Passive Sensing)
사용자의 개입 없이 백그라운드에서 수집되는 데이터로, 정신건강의 객관적 지표로 활용됩니다.
*   **가속도계 & 걸음 수:** 활동량 감소 -> 우울/무기력(정신운동지체) 징후.
*   **스크린 타임 & 잠금 해제:** 잦은 확인 -> 불안/초조. 야간 사용 -> 수면 장애.
*   **GPS (위치 변동성):** 생활 반경 축소 -> 사회적 위축(Social Withdrawal).

### 1.2 능동적 데이터: SAM (Self-Assessment Manikin)
언어적 개입을 최소화한 시각적 정서 측정 도구입니다.
*   **Valence (쾌-불쾌):** 찌푸린 캐릭터(1) ↔ 웃는 캐릭터(9).
*   **Arousal (각성-이완):** 졸린 캐릭터(1) ↔ 눈이 크고 가슴이 뛰는 캐릭터(9).
*   **UI 시각화:** 선착장의 날씨(Valence)와 파도 높이(Arousal)로 직관적 입력 유도.

## 2. NLP 및 감정 분석 파이프라인
사용자가 작성한 텍스트(미끼)를 로컬(On-device)에서 분석합니다.

### 2.1 LIWC (Linguistic Inquiry and Word Count) 기반 분석

| LIWC 카테고리 | 키워드 예시 | 정신건강학적 해석 (Dr. Mind) | 시스템 피드백 (Dr. Neuro) |
| :--- | :--- | :--- | :--- |
| **Absolutist** | 항상, 절대, 모두, 전혀 | 흑백 논리, 인지적 왜곡 가능성 | "생각의 유연성" 퀘스트 제안 |
| **Negative Emotion** | 슬픔, 화남, 짜증 | 부정 정서 표출 | 해당 정서 물고기(예: 복어) 출현 확률 증가 |
| **Cognitive Insight** | 깨달았다, 왜냐하면 | 치료적 통찰, 인지적 처리 | "지혜의 진주" 아이템 드롭 (보상 강화) |
| **Focus Past** | 했다, 였었다 | 반추(Rumination) 경향성 | 현재/미래 지향적 질문 제시 |
| **Focus Future** | 할 것이다, 만약 | 예기 불안(Anticipatory Anxiety) | "지금-여기" 호흡 퀘스트 제안 |
