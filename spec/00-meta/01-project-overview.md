# Project Overview: InsightMe (The Meta-Cognition Engine)

## 1. 프로젝트 정의
사용자의 일상 기록(수면, 감정, 사건)을 통해 무의식적인 행동 패턴과 핵심 가치를 발견하고, 뇌과학/심리학 기반의 코칭 질문을 통해 스스로 답을 찾게 하는 **'AI 기반 자아실현 파트너'** 서비스.

## 2. 융합 전문가 관점의 목표 (The "Why")

### 🧠 뇌과학적 관점 (Brain Science)
* **전두엽 활성화:** 감정과 충동을 '기록'하는 행위는 편도체(Amygdala)의 활성을 낮추고 전두엽(Prefrontal Cortex)을 개입시켜 감정 조절 능력을 키운다.
* **수면과 기억:** 수면 기록을 통해 렘수면(REM)과 비렘수면의 패턴을 파악, 기억 통합과 감정 정화의 최적 타이밍을 찾는다.

### 🧘 심리학적 관점 (Psychology)
* **자기 결정성 이론(SDT):** 자신이 무엇에 행복하고 화가 나는지 아는 것은 자율성(Autonomy) 회복의 핵심이다.
* **인지행동치료(CBT):** '상황-생각-감정-행동'의 고리를 파악하여 부정적 사고 패턴(인지 왜곡)을 수정한다.

### 💼 서비스 기획 관점 (Service Planning)
* **Hyper-Personalization:** 단순 통계가 아닌, "당신은 비 오는 날 오후 2시에 집중력이 떨어지는 경향이 있습니다"와 같은 초개인화된 맥락(Context) 정보를 제공한다.
* **Actionable Insight:** "화를 내지 마세요"가 아닌 "지금 산책을 5분 하면 코르티솔 수치가 낮아질 것입니다"와 같은 구체적 행동 제안.

## 3. 핵심 가치 제안 (Core Value Proposition)
1.  **Discover:** 나의 무의식적 트리거(Trigger) 발견
2.  **Understand:** 내 감정과 행동의 원인 이해
3.  **Optimize:** 나만의 최적 루틴(수면, 몰입) 구축
