# Glossary & Terminology

## 🧠 Core Domain (Psychology & Neuroscience)

| 용어 (Korean) | 용어 (English) | 설명 및 코드 변수명 컨벤션 |
| :--- | :--- | :--- |
| **자동적 사고** | `Automatic Thought` | 상황 발생 시 반사적으로 떠오르는 생각. CBT의 핵심 분석 대상. |
| **인지 왜곡** | `Cognitive Distortion` | 논리적 오류가 있는 사고 패턴 (예: 흑백논리). `distortionType` |
| **트리거** | `Trigger` | 감정이나 행동을 유발한 촉발 요인. `triggerSource` |
| **회피 행동** | `Avoidance Behavior` | 부정적 감정을 피하기 위해 하는 행동 (예: 폭식, 잠). `avoidanceAction` |
| **대처 행동** | `Coping Strategy` | 스트레스를 관리하기 위한 능동적 행동. `copingAction` |
| **신경 가소성** | `Neuroplasticity` | 반복된 행동으로 뇌 구조가 변하는 성질. 습관 형성 로직의 근거. |
| **몰입** | `Flow` | 시간 가는 줄 모르고 집중하는 최적 경험 상태. |

## 🛠 System Domain

| 용어 | 설명 |
| :--- | :--- |
| **Insight** | 데이터 분석을 통해 도출된 사용자의 패턴. `InsightCard` 형태로 UI 노출. |
| **Session** | 앱 사용 단위가 아니라, '하나의 감정/사건을 기록하고 회고하는 전체 과정'. |
| **Vibe** | 사용자가 느끼는 전반적인 분위기나 기분 상태 (Mood보다 포괄적). |
