# ZestinMe 앱 개요

## 앱 비전
개인의 행복과 수면 패턴을 기록하고 분석하여 더 나은 라이프스타일을 제안하는 개인화된 웰빙 앱

## 핵심 가치 제안
- **간편한 기록**: 10초 만에 완료되는 감정/수면 기록
- **개인화된 인사이트**: 나만의 행복 패턴과 최적 수면 시간 발견
- **지속적인 개선**: 7일 챌린지를 통한 습관 형성 지원

## 주요 기능 영역

### 1. 행복 기록 (Happy Record)
- 행복한 순간을 간편하게 기록
- 감정 강도, 태그, 상황 등을 포함한 상세 기록
- 시각적 타임라인으로 기록 관리

### 2. 수면 기록 (Sleep Record)  
- 취침/기상 시간 기록
- 수면의 질과 하루 컨디션 평가
- 개인별 최적 수면 패턴 분석

### 3. 코칭 시스템
- AI 기반 개인화된 질문과 가이드
- 감정 상태에 따른 맞춤 조언
- 7일 챌린지를 통한 목표 달성 지원

### 4. 분석 및 통계
- 주/월간 감정 패턴 분석
- 수면과 컨디션의 상관관계 분석
- 개인화된 인사이트 제공

## 타겟 사용자
- **주요 타겟**: 20-40대 직장인 및 대학생
- **니즈**: 스트레스 관리, 수면 개선, 자기계발에 관심이 높은 사용자
- **행동 특성**: 간편하고 직관적인 앱을 선호하며, 데이터 기반 인사이트를 가치있게 여김

## 기술 스택
- **프론트엔드**: Flutter (iOS/Android 크로스 플랫폼)
- **백엔드**: Firebase (Authentication, Firestore, Functions)
- **아키텍처**: Clean Architecture + BLoC 패턴
- **상태관리**: Provider/BLoC

## 성공 지표
- **참여도**: 일일 능동 사용자(DAU) 유지율
- **완주율**: 7일 챌린지 완주율 70% 이상
- **만족도**: 앱스토어 평점 4.5 이상 유지
