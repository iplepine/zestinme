# 기능 맵

## 현재 핵심 표면

| 영역 | 사용자 가치 | MVP 상태 |
| --- | --- | --- |
| 온보딩 | 첫 기록을 만들고 홈으로 진입 | 필수 |
| 홈 | 오늘 상태와 지금 할 일 확인 | 필수 |
| 컨디션 체크인 | 에너지/회복 중심 빠른 기록 | 필수 |
| 회복 로그 | 수면과 아침 회복감 기록 | 필수 |
| 타임라인 | 날짜별 기록 재확인 | 필수 |
| 설정 | 테마, 언어, 기본 정보 관리 | 유지 |

## 데이터 기준

| 데이터 | 역할 | 기준 |
| --- | --- | --- |
| `ConditionRecord` | 컨디션 체크인 | 현재 핵심 |
| `SleepRecord` | 회복 로그 | 현재 핵심 |
| 개입 이벤트 | 제안 결과 반영 | MVP 필요 |
| `EmotionRecord` | 과거/래핑 흔적 | 레거시 주의 |

## 후순위 후보

- 취침 전 wind-down 개입
- 집중 블록 타이머
- 알림 기반 개입
- 패턴 기반 개인화
- 외부 데이터 연동
- 리포트/공유용 요약
